01 Getting Started

You must have WordPress installed and working to start using this theme.

If you need help getting it installed, go to http://codex.wordpress.org/Getting_Started_with_WordPress.  They cover everything from the 5 minute WordPress installation to hosting to various page settings.

Installing QuickStep

You have two options for installing the theme:

  • FTP - Log into your site using your favorite FTP program and go your WordPress installation’s /wp-content/themes directory.  Upload the unzipped ‘quickstep’ folder into this directory.
  • WordPress Admin Upload - Log into the admin area of your WordPress site.  Go to Appearance -> Themes.  On this page click ‘Install Themes’ and select upload.  Then choose the zipped QuickStep theme.

After you’ve finished uploading your theme, navigate back to Appearance -> Themes.  You should now see the QuickStep theme as an option.  Click ‘Activate’ underneath the theme and you’ve finished installing QuickStep.

One Page Scrolling

Please make sure that your home page is not set to a ‘Static Page.’  To check this, go to Settings -> Reading and make sure the radio button ‘Your Latest Posts’ is selected.

Standard Website

If you’d rather have your content load on separate pages like most web sites, set your home page to a static page of your choice by going to Settings -> Reading.  Also uncheck the ‘One Page Enabled’ option in your Theme Options panel.

Installing demo content (Recommended)

We highly recommend that you install the demo content for QuickStep.  It will make it much easier to understand the full functionality of the theme if it’s your first time using it.

To install the demo content:

  • Go to Tools -> Import in your WordPress admin area
  • Click on WordPress (you may need to install the importer plugin if you have not used it before)
  • Click Choose File and select the demo content you’d like to import
  • If you want to import the theme images as well, make sure you tick Download and import file attachments.

Depending on the speed of your internet connect this may take a while, so please be patient.

At this point, you should go to Appearance -> Theme Options to set the style for your web page. Even if you want to use the theme defaults, you will need to click Save Options on this page to apply them to your site.

Installing Recommended Plugins

The themes used in the demo come pre-packaged with QuickStep. After you’ve installed and activated the theme, you should see a notice that asks you to install the plugins. If you’d like to install them, simply click the link in the message to install and activate each plugin.

If you don’t need the plugin, just dismiss the notice. None of the plugins are required for the theme to work, but they do add some nice functionality.

Contact Form 7

This plugin allows you to easily create and modify your own contact form.  You can easily add and remove fields.  When you want to add it to your page, paste the shortcode provided by the plugin into your page and your down.

For full documentation on this plugin, please visit http://contactform7.com/docs/

Twitter Widget Pro

This plugin allows you to enter a username and retrieve their latest tweets on your site.  For full documentation, please visit http://wordpress.org/extend/plugins/twitter-widget-pro/.

 

 

02 Theme Options

The site’s theme options are all available by going to Apperance -> Themes.  Here you can easily change the style and customize your theme.

General

This tab houses options for uploading your logo, favicon, inserting custom code, and changing the page scroll settings.  A long list of options is available to you to control how your page shifts from one page to another and how fast.

Body

The body tab controls the basic style of the content area of your site.  You can pick your site’s accent color and backgrounds here.  If individual page backgrounds are chosen, they will override these settings so that you can have unique page attributes.

Here you can also select a custom font for your site.  There are 500+ fonts that are regularly updated uses Google’s API.

Header

The header comes with a number of options depending on how you’d like your home page to look.

  • Static will keep the header and navigation at the very top of your site at all times.
  • If fixed is selected, the header will stay at the top of your screen at all times.  This provides easier navigation for your users if you have a large site.
  • Fade In will hide the header at the top of the page.  After the user scrolls down, the header will appear.  You can also set the distance scrolled until the header appears under this tab.
  • Hidden simply removes the menu from the site.  This is a good option if you have a smaller site or you’d like to use buttons to navigate the site (for example, “Click Here to Continue!”)

Footer

The footer tab lets you control the text and colors of the footer.  The widget area refers to the widgets you drag and drop into the footer, while the actual footer is the smaller area underneath containing copyright information.  The footer text area can contain HTML and shortcodes if you’d like.

Portfolio & Blog

Here you can control various elements related to each of this page styles.

Social

Here you can add in links to your social media profiles.  Once you’ve added the links here, you can display the social icons inside a page using the [social] shortcode.

03 Pages

Creating pages is a breeze with QuickStep.  First, navigate to Pages -> Add New.  Then add in your various page content, images, and shortcodes.  When you’re finished click Publish and your page will be created.

There is no specific home page in QuickStep.  You can add sliders and background images to any page.  To change the order of the pages displayed, change the Order number in the Attributes section when editing a page.  Pages will show up in ascending order on the front end.  If an order is not entered, the pages will show in chronological order.

Templates

When editing a page, you have a number of templates to choose from.  When editing a page, you can select a template from the drop-down menu under Page Attributes.

Default Template

The default template is a page with left-hand content and a right-hand sidebar.

Blog

The blog template will show your most recent blog posts with a right-hand sidebar.  The default posts per page is 10 in WordPress.  The online demo theme uses 5 posts per page.  To change the number of posts per page, go to Settings -> Reading and enter the number of posts you’d like per page in the Blog posts show at most field.

Full Width

This layout has no columns.  It uses the full width of the content area.  This is ideal for most pages, especially the home page.

Left Hand Sidebar

This page is the same as the default layout, but with the sidebar on the left-hand side.

Portfolio 3 Columns

This template displays all your portfolio items with a maximum of 3 items per row.

Portfolio 4 Columns

This template displays all your portfolio items with a maximum of 4 items per row.

Portfolio 5 Columns

This template displays all your portfolio items with a maximum of 5 items per row.

Team Page

Displays all your Team Members in a four column layout.

 

Image sizes for all pages are automatically created when uploading an image.

Page Settings

Each page has its own custom settings you can use when editing a page.  On the right side, you’ll see General Page Settings.  Here you can:

  • Change the name of the page
  • Add a subtitle to show beneath the header
  • Remove the page title (for example, you may not want to show ‘Home’ for the homepage)
  • Remove the page from navigation.  The page will remain on your page, but won’t show up in the top navigation menu.

Below the editor, you will find Page Style.  This lets you change the background, font colors, and page height.  Below are a few quick tips for individual page settings.

  • Every page can have its own background image, but we don’t recommend using too many.  Using many large background images may slow down page loads for slower internet connections.
  • Using the Cover option under Page Background Repeat is a great CSS3 technique to make sure the background is stretched to fit the screen without skewing the image.  If you do have a full page background though, it is a good idea to make sure it is naturally large enough.  Some browsers may perform slower when stretching a small image to fit a larger screen.
  • Page Height will set a page to a specific height.  You can use this if you’d like to make sure that an entire background is seen or add space between the next page.  As a warning, this will remove the responsiveness of the page height.

Page Attributes allows you set the template, order, and parent of the page.  QuickStep support sub-pages and sub-menus, but does not support third-level items.  Pages are displayed directly underneath their parent page.

Posts

Posts are displayed in chronological order on your blog page template.  To create a post go to Posts -> Add New.

QuickStep uses the Feature Image to display post images.   You can also set the categories and tags for the post using the edit post page.

When you’re finished, click Publish and your post will automatically be added to your blog page.

Linking and Menus

Linking

QuickStep uses a quite advanced method to link pages if you’re using the “One Page” option.

The top navigation will scroll to any content area if it’s currently on the page.  If it’s not on the page, it will attempt to load it dynamically through AJAX at the bottom of the page.

All links on the page will load through AJAX into the bottom of the page if the link is on your site.  To slide to certain sections on your one page template, you can use the link and button shortcodes.

To open up a page and have it scroll down to a certain section, add ?slide=PageSlug to the end of the URL.  For example, http://themeluxe.com/themes/quickstep/?slide=contact to slide to the contact page on page load.  Take a look at the button short code documentation to learn how to find a page slug.

To load a page dynamically into the bottom of your page when your page loads, add ?load=EncodedPageURL to the end of the URL.  This URL must be encoded first so that special characters are removed and it can be passed to the server as a parameter.  To encode a URL quickly, you can use a site like http://encodeurl.com.  Copy and paste your URL and click ‘Encode.’  For example:  http://themeluxe.com/themes/quickstep/blog/portfolio/oxygen/ becomes http%3A%2F%2Fthemeluxe.com%2Fthemes%2Fquickstep%2Fblog%2Fportfolio%2Foxygen%2F.  Then copy and paste this encoded URL to the end of your domain URL:

http://themeluxe.com/themes/quickstep/?load=http%3A%2F%2Fthemeluxe.com%2Fthemes%2Fquickstep%2Fblog%2Fportfolio%2Foxygen%2F

This can be useful if you want users to open a post directly instead of searching for it on your one page theme.  As another option, you can also link users directly to the page ( http://themeluxe.com/themes/quickstep/blog/portfolio/oxygen/ ).  The navigation will automatically be changed through Javascript to refer users back to the main page when clicked.  Since this is all done through Javascript, you don’t have to worry.  Google and other search engines still read the links like a normal site so you can get all the great SEO benefits of a non-one-page web site.

Menu and Page Setup

The QuickStep menu works out-of-the-box, so there’s no need for any initial setup.  The menu and the pages are ordered by the ‘Menu Order.’  To change this click Pages and hover over the item you’d like to change.  Click Quick Edit and change the order number to the order you’d like it to appear.  On a side note, you can turn off comments for any page in this area.

You can also setup custom menus by going to Appearance -> Menus.  Add the items that you’d like to appear in the menu and make sure that you select the Theme Location of the menu so that it will show up.

For advanced options, click Screen Options at the top right of your Dashboard.  Link Target will let you open a link in a new tab or window.  If you’d like to separate pages from your one page scrolling site select CSS Classes and add ‘separate’ to the CSS classes field of the link you’d like to separate.

If you’d like to hide a page from your one page scrolling site, simply set it as a third-level menu item.  Make sure to give it a parent page and give that a parent page.  These parent pages can be set to ‘Draft’ by clicking Quick Edit if you don’t want them showing up on your site.

04 Custom Post Types

Custom post types help you organize and display specific types of content.  QuickStep comes with 3 custom post types at your disposal.  You can find each of them in the left-hand sidebar in your WordPress admin area.

Features

This post type helps you display and promote your brand through an image, tagline, and description.  To add a feature, navigate to Features -> Add New.  For the feature image you can either upload an image to use for the Featured Image, or use an icon from the icon set packaged with QuickStep.  If an icon is set, the image field will be ignored.

To see a full list of the font icons available and the associate letter, please visit http://thedesignoffice.org/project/modern-pictograms/.

Once you’ve created a feature, you can copy the shortcode from the Features page and paste it into any page.

 

Sliders

QuickStep comes with FlexSlider and is touch ready for mobile devices.  You can add sliders anywhere around your site and there is no limit on the number you can use.  To create a slider go to Sliders -> Add New.

Here you can upload images and add content to slides.

  • Every slide must have an image.
  • The content section can use HTML and shortcodes
  • To re-order the slides, simply drag the title to the position you’d like.

To change slide settings, animation, and speed use the Slider Settings to the right of the slides.  Each slider can have its own settings.

When you’re finished, click Publish to create your slider.  You can then grab your slider shortcode by clicking Sliders and copying the shortcode from the respective column.  You can paste this shortcode into any page to view your slider.

Team

To create a new team member, go to Team -> Add New.

For the title, enter the person’s name.  You can a short description about the team member in the editor.  To add a job title for the member, type the title in under Member Settings.  The featured image will be used as the team members picture.  The picture is cropped when uploaded to 180 px by 180px.

If you want to add social links for the team member, paste the link in Social Links and you’re all set.

When you’re finished click Publish.  Make sure one of your pages is using the ‘Team Page’ template to display your team members.

Portfolio

To create a new project, navigate to Portfolio -> Add Item.

To set the image for the project item, click Set featured image and select the image you’d like to upload.  Images are automatically cropped to the following sizes:

  • 320px by 240px for the three column portfolio
  • 240px by 180px for the four column portfolio
  • 192px by 144px for the five column portfolio
  • 630px by 230px for the project page

We recommend that you use a larger image.  Your image will automatically be cropped and scaled to fit these dimensions.

Project information can be set below the editor box, including Client Name, Website, and Short Description.

When you’re finished creating your project item, click Publish.  Make sure one of your pages has a ‘Portfolio’ template selected to display your project items.

Post Formats

QuickStep supports a number of post formats.  These can be used in both blog posts and portfolio items.

For regular posts, select the format to the right of the editor when editing a post.

  • Standard - This is the default for all posts.  It uses your featured image, title, and content to display a post.
  • Aside – Asides are displayed without titles and without featured images.
  • Gallery - The gallery option will display a box below your editor where you can copy and paste your slider code.  (Go to Sliders from your left-hand dashboard menu to get this code).  You can also paste any other code you’d like to use directly into this box.
  • Link - To use a link formatted post, enter the URL in the box below the editor.  You can also choose whether you want it to open in a new browser tab or load separately (non-AJAXed) from your main scrolling page.
  • Video - Video format displays a field below the editor where you can paste your video code.  You can paste Youtube, Vimeo or any other iframe/embed codes directly into this box.  For example, to obtain a Youtube embed code, go to the video you’d like to use and click “Share” then click “Embed.”  You’ll then be provided with a code you can copy and paste into this field.

To use a video or gallery for your portfolio items, simply copy and paste your code into the Custom Project Code field below the editor.  You can use shortcodes (such as sliders), embed, iframe, or HTML codes here.  If you leave this field blank, your portfolio item will default to the featured image.

05 Shortcodes

Shortcodes are a great way to add lots of functionality and design to your site without prior knowledge of code.

Columns

Columns help organize your content in a meaningful way.  When adding columns, make sure that you add ‘_last’ to your final column shortcode.  For example:

[one_third] First Column [/one_third]  [one_third] Second Column [/one_third]  [one_third_last] Third and Final Column [/one_third_last]

 

[one_half] Your content here [/one_half]

[one_third] Your content here [/one_third]

[two_third] Your content here [/two_third]

[one_fourth] Your content here [/one_fourth]

[three_fourth] Your content here [/three_fourth]

[one_fifth] Your content here [/one_fifth]

[two_fifth] Your content here [/two_fifth]

[three_fifth] Your content here [/three_fifth]

[four_fifth] Your content here [/four_fifth]

 

Buttons

You can use buttons to draw attention, create links, and as a way to navigate your site.

[button] Button Text [/button]

You can also use the following attributes to style your button:

  • link
    • Options:  Any link
    • Example:  [button link="http://themeforest.com"] Go to ThemeForest [/button]
  • size
    • Options:  small, medium, large
    • [button size="large"] This is a large button [/button]
  • align
    • Options: left, right
    • [button align="right"] Right aligned Button [/button]
  • slide
    • Options: Page Slug
    • [button slide="contact"] Slide to Contact Page [/button]
    • To find the page slug, go to Pages and click Quick Edit on the page you’d like to find the slug for.
  • color
    • Options: inverse
    • [button color="inverse"] Inverse Color Button [/button]
    • The regular button color is the color chosen in the admin panel.  An inverse colored button starts as your theme color and switches to the normal button color when hovered.

Icons

QuickStep comes packaged with Modern Pictograms icons.  You can display an icon by using the icon shortcode tag.  The following will display a camera icon.

[icon]A[/icon]

You can use the following attributes to style your icon:

  • size
    • Options:  Any Number (in pixels)
    • Example:  [icon size="80"]c[/icon]
  • color
    • Options:  Any basic color (red, white, blue, etc.) or any hex color value (#000000)
    • Example:  [icon color="red"]E[/icon]
    • Example:  [icon color="#00325b"]8[/icon]
  • align
    • Options: left, right
    • Example: [icon align="left"]Y[/icon]

To view a full list of the icons you can use and their characters, please visit http://thedesignoffice.org/project/modern-pictograms/.

Tabs

You can use tabs to better display and organize content on your site.  To use tabs, you must start and stop the tabs with the [tabgroup] and [/tabgroup] shortcode, respectively.  You can then use the [tab] shortcode to create a new tab and end the tab’s contents with [/tab].

The tabs shortcode uses an attribute called title.  This is used for the tab name.  It also uses an id tag.  This can be whatever you choose, but it must not be identical to any others and must not contain any special characters or spaces.

[tabgroup]
[tab title="Tab 1" id="t1"] Tab 1 Content [/tab]
[tab title="Tab 2" id="t2"] Tab 1 Content [/tab]
[tab title="Tab 3" id="t3"] Tab 1 Content [/tab]
[/tabgroup]

Toggles

You can use toggles to group content into expandable sections on your site.  To use toggles, you must start and stop the toggles with the [togglegroup] and [/togglegroup] shortcode, respectively.  You can then use the [toggle] shortcode to create a new togggle and end the toggle’s contents with [/toggle].

The text on the toggle button is determined by its title attribute.

[togglegroup]

[toggle title="Button text One"]
Toggle 1 Content.
[/toggle]
[toggle title="Button Text Two"]
Toggle 2 Content.
[/toggle]
[toggle title="Button Text Three"]
Toggle 3 Content.
[/toggle]

[/togglegroup]

Blockquote


Blockquotes emphasize blocks of text to readers.  You can use the line below or you can use the editor button with a picture of a quote.
<blockquote> Your blockquote text goes here. </blockquote>


Info Messages

Info messages can be used around your site to inform the visitor upon success or failure of an action.

It accepts the attribute type, which determines the type of message coloring and icon displayed.

[info type="success"]Success message[/info]

[info type="error"]Error message[/info]

[info type="warning"]Warning message[/info]

[info type="info "]Info message[/info]

Accents and Highlights

Accents and highlights are used to draw attention to specific words or text.  An accent changes the color of the word to your sites accent color selected in Theme Options.  A highlight does the same but changes the background color behind the text.

This word is [accent]accented[/accent] with your site’s color.

This word is [highlight]highlighted[/highlight] with your site’s color.

By default highlight uses your theme’s accent color, but you can also use the following attributes to modify the highlight.

  • color
    • Options:  Any basic color (red, white, blue, etc.) or any hex color value (#000000)
    • Example:  [highlight color="#000000"] Black Highlighted Text [/highlight]
  • opacity
    • Options:  1-100
    • [button opacity="70"] A highlight with an opacity of 70% [/button]

Google Maps

This shortcode displays a Google Maps of the location inside your page.  For the demo content, it’s used on the Contact Us page.

To use it, go to https://maps.google.com/ and find you location first.  Once you’ve found it, use the Link button (to the right of Print and Get Directions) to copy the link of the location.  Copy and paste this between the [googlemap] shortcode tags.

[googlemap]https://maps.google.com/maps?q=Los+Angeles,+CA,+United+States&hl=en&sll=33.748995,-84.387982&sspn=0.477299,1.056747&oq=los+a&hnear=Los+Angeles,+California&t=m&z=10[/googlemap]

Features

Social

The social shortcode displays your social media links.  To add social media links, visit Apperance -> Theme Options to add and edit URLs for your social profiles.  Once set, the shortcode below will add them to your page.

[social]

Links

Links are similar to buttons.  They were added to the theme to allow users to slide to content around the site.

[link] Link Text [/link]

You can also use the following attributes to style your button:

  • link
    • Options:  Any link
    • Example:  [link link="http://themeforest.com"] Go to ThemeForest [/link]
  • align
    • Options: left, right
    • [button align="right"] Right aligned Button [/button]
  • slide
    • Options: Page Slug
    • [link slide="contact"] Slide to Contact Page [/link]
    • To find the page slug, go to Pages and click Quick Edit on the page you’d like to find the slug for.
  • target
    • Options: _blank, _self
    • [link target="_blank" link="http://themeforest.com"] Open a new browser tab[/link]
  • separate
    • Options: true
    • [link separate="true"  link="/post-name"] Change the current page without AJAX[/link]
    • This allows you to separate individual pages so that they aren’t loaded using AJAX into your main page.

06 Theme Files

This section talks about the various theme files that make up QuickStep.  If you want to modify or add to any of the code, start here.  Otherwise, feel free to skip this section.

Javascript

  • js/customjs.php - Contains the custom functions written for QuickStep and custom user theme options
  • js/jquery.easing.1.3.js - jQuery easing script for animation and transitions
  • js/jquery.flexnav - Makes the navigation responsive for smaller screens
  • js/jquery.flexslider - Flexslider script for sliders
  • js/jquery.isotope.min - Isotope file for portfolio filtering
  • js/jquery.js - Javascript library that provides a fast and flexible way to implement new functions
  • js/jquery.prettyPhoto - PrettyPhoto functionality to allow images to be zoomed in a modal box
  • js/jquery.scrollTo-1.4.2-min.js - A script that adds scrolling features to the page
  • js/jquery.showLoading.js - Overlays a loading image on a div when an AJAX function is called
  • js/jquery-ui-1.8.20.custom.min - jQuery UI library for tabs, toggles, etc.

 

CSS

  • style.css - This is the base theme stylesheet.  The majority of styling is in here and broken into sections by comments.
  • css/customcss.php - Contains the user theme option styling.
  • css/dark.css - Enabled if the user chooses a ‘dark’ base style.  It is mostly for UI elements and other shaded areas.
  • css/flexnav.css - Navigation styling and responsive media queries for smaller screens
  • css/flexnav_ie8.css - Fixes the navigation queries for older versions of Internet Explorer
  • css/flexslider.css - Basic styling for the slider
  • css/foundation.css - Reset styling for the page, HTML5 elements, and responsive media queries for columns
  • css/prettyPhoto.css - Styling for PrettyPhoto image container
  • css/social_foundicons.css - The social icon font used with the theme.  Currently only used for team member social links.
  • css/social_foundicons_ie7.css - Fixes for the social icon font set for IE 7
